
# ICON SCORE Support

## Features
A VSCode extension to deveop ICON SCORE.
Helping ICON SCORE develop. SCORE can be developed and easily deployed on the ICON network. We plan to offer more features in the future.

## Requirements
VSCode v 1.31.1+
